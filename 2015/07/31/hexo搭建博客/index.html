    • 1. 安装git

    mac上安装git的方法有很多，我使用的方法是：

    1. 在AppStore里面下载安装 Xcode（需要AppleID）；
    2. 打开Xcode–>Preferences（快捷键 Command + ，（逗号））点击Downloads选项，下载Command Line Tools。
    3. 打开终端，输入命令：git –version，查看当前git版本。

    注意：在第二点中，如果Downloads里没有Command Line Tools的选项，则可以直接下载它的安装包或者打开Xcode–> Open Developer Tool->More Developer Tools进入下载页面，点击下载Command Line Tools，安装即可。

    2. 安装node.js
